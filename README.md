# ZTFalloutRPG
## A post-apocalyptic Discord Bot

### Author: Tommy Tran (ZombieTommie)
### Date: 5/5/2026

This is a bot for Discord to enable users to play Fallout the RPG TTRPG. Adding abilities to roll dice, gain and spend AP, track character, track initiative, and more to come as this is developed. 

I want to make this a simple bot first to allow simplified Play by Post on Discord. 

Written in python, using the discord.py
Database will be SQLite, possible integrate into PostgreSQL.

Logs are written to `logs/discord.log` when running locally or through Docker Compose.

## Features:
- Hello
- Roll 2d20 dice
- Manage AP for GM and Player Pools
- Caps Management
